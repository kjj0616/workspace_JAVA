## JRE ?? JDK ?? ##
  자바로 만들어진 프로그램의 경우, 
  사용자들이 구동시키 위해선 현재 운영체제에 JRE (Runtime Environment) 라는걸 
  다운받아서 설정이 잡혀져 있어야 한다.
  반면, 자바로 프로그램을 만들기 위해선, JDK (Development Kit) 라는걸 설치해야한다.

## 컴파일, 인터프리트 ##
 소스파일             ==>                   프로그램                     ==>       컴퓨터
              컴파일 by 컴파일러  in JDK           인터프리트 by 인터프리터 (≒JVM)  in JRE
               개발자가 해야되는 부분                    사용자가 해야되는 부분
                (만들기)                                                           (실행하기)            

## JDK는 어디서?? ##
   oracle 공홈에서 받아야 한다.    java.oracle.com
   우측 링크에  Java SE 8 Update 181  클릭 후 이동 →   버전 찾아서,  운영체제에 맞게 JDK 설치 
   ( ※ 실제 자바의 버전은 10까지 나와있지만,  9가 성공하지 못했고,  10은 개발중인 버전이기에 
       사용자측 JRE를 8버전으로 제공하고 있다. 그에 맞춰 JDK 버전을 잡는게 호환성면에서 좋다.)

## 개발툴 ?? ##
   프로그램을 좀 쉽게 만들어내기 위해서, 필수적인 건 아니지만 개발툴을 이용해서 프로그램을 만들어나간다.
   Java 프로그램 만들어낼때 사용되는 개발툴의 종류는 여러가지이다. Eclipse / IntelliJ / editplus 등등 다양하게 존재한다.
   
   실제로 프로그램을 만들때야 있는게 좋지만, 문법 공부를 하는데 있어서는 필요하지 않을수도 있다.
   (※현재 1개월차엔 editplus / 2개월차부터 eclipse , intelliJ 둘다 사용하는 법을 익힐 계획이다.)

## 툴이 없다면 프로그램을 어떻게? ##
    운영체제에서 제공하는 글자 작성가능한 기본 텍스트편집기만 있으면 된다. 
     
## 프로그램은 어떻게 만들어?? ##
 
 step1:  툴이 없다면 , 메모장 편집기를 하나 열어서 파일명은  편하신대로 설정하고   파일확장자를     .java  로 설정.
   작성된 코드를 토대로 프로그램을 만들려면 compile 을 진행해야 함. 
   (JDK가 있어야만 가능한 작업 / 개발을 할때 사용되는 프로그램은 jdk1.X.XXX/bin  폴더내에 다 깔려있음)
   개발툴이 있다면, 좀 편하게 작업을 할 수 있겠지만, 그게 아니라면 수동으로 만들어야 한다.
  
 
  step2:  cmd 를 키신 후 , 소스 코드가 있는 곳까지  cmd 상 에서 cd 명령어를 이용해서 이동 (그 외의 명령어는 help 를 치면 확인 가능)
   소스코드를 컴파일을 진행해야 프로그램이 만들어지는데    javac  파일명 (확장자포함)   을 작성하면 된다.
   만들어지는데 성공하면 
          아무 것도 출력되지 않고, 소스파일이 있던 곳에 만들고자했던 프로그램이름.class 가 생성
   그게 아니라면 
          왜 실패했는지 원인이 출력된다.
  (
     설정이 다 안잡혀 있어서, javac 라는 명령어가 여기서 작동을 안함 / 추가설정이 필요함.
     운영체제 환경변수를 설정해야 한다.  시스템속성 을 열어서 window+pause / 설정 변경 / 고급 /  환경변수 클릭
     1.  JAVA_HOME 을 등록 :
             시스템변수 새로만들기 / 변수명을 JAVA_HOME  디렉토리 찾기 후  JDK 설치디렉터리를 선택
     2. Path 수정 :  %JAVA_HOME%\bin  추가
     재설정을 했다면 step2로 ..
  )
 
  step3: 만들어진 프로그램 ( .class )을  실행을 시키려면,   java 프로그램명(확장자없이)
