 # VHD 설정 #
  : 앞으로 6개월간 사용할 운영체제를 하나더 설정
  step.1 
   window+r → diskmgmt.msc → 드라이브 우클릭(d: 혹은 e:) 후 볼륨축소
   → 102400 (단위는 MB) 축소 → 독립으로 만들어낸 볼륨 우클릭 → 새단순볼륨 선택
   → 다음..다음..다음.... → 포맷 까지 되고 나면 완료.

   만들어진 드라이브로, 받은 win10.zip 을 압축해제 (바로 루트에 푸세요)

  step.2  
   설치프로그램에서 cmd 검색 → 관리자권한으로 실행
   → bcdedit /copy {current} /d "ncs_16"  (복사됬다는걸 확인)
   → 복사된 {.....} 내용을 복사 (드래그 후 우클릭)
    / {ca5a5b10-536d-11e8-94d2-956642373bd9}
   → bcdedit /set {ca5a5b10-536d-11e8-94d2-956642373bd9} device VHD=[G:]\win10.vhd
   → bcdedit /set {ca5a5b10-536d-11e8-94d2-956642373bd9} osdevice VHD=[G:]\win10.vhd
  
   → (현재 이 PC의 IP를 확인해두자 / ipconfig ) 

  step.3
    window+r  →  msconfig  → 부팅 으로 이동  → 목록에 방금 만든 부트모드가 있는 확인
    → 혹시 시간이 0으로 설정되있다면, 선택할수 있는 시간을 적당하게 설정
  
  step4.
   재부팅 후 새로 등록한 모드로 부팅 이후, (기본 암호는 P@ssw0rd)
      ?? 인터넷이 안된다(IP 할당을 못받는다면,  )
        설치목록에서   네트워크 검색(네트워크 상태)   → 어댑터 옵션 변경 → 플러그 빠진 모양 우클릭 속성 
        → 인터넷 프로토콜 버전 4 를 클릭 편집 → 자동말고 직접 기입 →
        아이피는 원래 본머신의 Ip를 기입 / 서브넷마스크 255.255.255.0  / 기본게이트 192.168.10.254
        기본 DNS   168.126.63.1    보조 DNS   168.126.63.2 

      ?? vhd 가 들어있는 디렉터리를 숨기자
      window+r → diskmgmt.msc  → vhd 넣어둔 볼륨 우클릭  → 드라이브 문자 및 경로 변경   → 문자 제거
